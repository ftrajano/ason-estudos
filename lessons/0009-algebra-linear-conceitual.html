Base, Independência Linear e Dimensão

A questão 24 — a última lacuna de Matemática. Conceito, não conta.

No simulado, autovalores e determinantes (a conta) você acertou; a Q24 era pura teoria: o que é base, o que é independência linear, qual a dimensão. São poucas ideias, e elas caem em quase toda prova como questão V/F. Vamos travá-las.

Três ideias que resolvem tudo

ConceitoO que é
Linearmente independente (LI)A única combinação \(c_1v_1+\dots+c_kv_k=0\) é a trivial (todos \(c_i=0\)). Se houver outra, é LD.
BaseUm conjunto que é LI e gera o espaço. Em \(\mathbb{R}^n\), uma base tem exatamente \(n\) vetores.
DimensãoO número de vetores de uma base. \(\dim(\mathbb{R}^n)=n\); \(\dim(P_n)=n+1\).

★ O teste que mata 90% das questões: \(n\) vetores em \(\mathbb{R}^n\) são LI (e formam base) se, e só se, o determinante da matriz que eles formam é \(\neq 0\).

\[ \det \neq 0 \;\Leftrightarrow\; \text{LI} \;\Leftrightarrow\; \text{base de } \mathbb{R}^n \qquad\qquad \det = 0 \;\Leftrightarrow\; \text{LD} \]

Sua questão 24, destrinchada

Afirmação 1: \( \{(1,3,5),(3,1,7),(5,3,13)\} \) é base de \( \mathbb{R}^3 \)? Calcule o determinante: dá \( 0 \) → são LD → não é base. (F)

Afirmação 2: \( P_n \) tem uma base com \( n+1 \) vetores de grau \( n \)? Sim — \( \dim(P_n)=n+1 \), e dá para escolher uma base toda de grau \(n\) (ex.: para \(P_2\), \( \{x^2,\ x-x^2,\ 1-x^2\} \) gera \(P_2\)). (V)

Afirmação 3: se \( \{v_1,v_2,v_3\} \) é LI, então \( \{v_1,\ v_2-v_1,\ v_3-v_1\} \) é LI? Sim — subtrair \(v_1\) é uma operação reversível, não cria dependência. (V)

Sequência: F – V – V (letra e). ✓

Por que a afirmação 3 é V: se \( c_1 v_1 + c_2(v_2-v_1) + c_3(v_3-v_1) = 0 \), agrupando vira \( (c_1-c_2-c_3)v_1 + c_2 v_2 + c_3 v_3 = 0 \). Como os \(v_i\) são LI, todos os coeficientes são 0 → \(c_2=c_3=0\) e então \(c_1=0\). LI preservada.
